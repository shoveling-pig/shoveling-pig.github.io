## Summary
> A keyboard shortcut is a series of one or several keys that invoke a software program to perform a preprogrammed action.

## Windows

### 한줄 복사 
`Shift + Alt + (↑, ↓)`

### 한줄 이동
`Alt + (↑, ↓)`

### 한줄 삭제
`Ctrl + Shift + K`

### 한줄 선택
`Shift + (↑, ↓)`

### 들여쓰기 & 내어쓰기
`Ctrl + ([, ])`

### 선택 라인으로 이동
`Ctrl + G + line_number`

### 폰트 사이즈 줄이기 & 키우기
`Ctrl + (+, -)`
